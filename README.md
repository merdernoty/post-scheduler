# Post Scheduler

Automation pipeline for generating, moderating, scheduling and publishing LinkedIn posts.

This repository contains the code and configuration for a LinkedIn content bot that generates text and images, presents drafts for moderation via Telegram, and publishes approved posts on a schedule.

## Privacy Policy

After enabling GitHub Pages for this repository, the privacy policy will be available at:

https://merdernoty.github.io/post-scheduler/privacy.html

### Enable GitHub Pages

1. Go to the repository Settings.
2. Open the **Pages** section.
3. Under **Source**, choose **Deploy from a branch**.
4. Select the **main** branch.
5. Choose the `/docs` folder as the source.
6. Save the settings.
7. Use the generated GitHub Pages URL as the LinkedIn Privacy policy URL.
