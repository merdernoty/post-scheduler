Privacy Policy

This application is a personal automation tool used to generate, review, schedule and publish LinkedIn posts.

Data Collected

The application may store generated post text, image URLs, scheduled publication times, Telegram user IDs and LinkedIn authorization tokens.

How Data Is Used

Data is used only for generating, moderating, scheduling and publishing LinkedIn content authorised by the user.

Data Sharing

This application does not sell or share personal data with third parties. Data may be sent to API providers required for text generation, image generation, Telegram moderation and LinkedIn publishing.

Data Deletion

The user can request deletion of stored data by contacting the application owner.
